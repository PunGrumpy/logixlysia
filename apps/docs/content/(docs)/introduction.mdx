Logixlysia is a logging plugin for [Elysia.js](https://elysiajs.com/) that provides a simple and flexible way to log requests and responses in your Elysia applications. Powered by [Pino](https://github.com/pinojs/pino), it delivers high-performance structured logging while remaining easy to use and highly configurable.

## Why use Logixlysia?

Integrating logging into your Elysia application can help you monitor and debug your application more effectively. Logixlysia provides a simple and consistent way to log requests and responses, making it easier to track down issues and understand how your application is behaving.
